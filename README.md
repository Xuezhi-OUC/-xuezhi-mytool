# xuezhi-mytool 🛠️

个人 Git 版本管理工具集，适用任何项目。

## 文件

| 文件 | 用途 |
|------|------|
| `SKILL-GIT.md` | Git 技能速查卡 — 初始化 / 提交 / 标签 / 回档 / 分支 / 常见问题 |
| `git-manager.sh` | 交互式管理脚本 — 菜单驱动，一键完成版本管理 |

## 快速开始

```bash
# 克隆到本地
git clone git@github.com:Xuezhi-OUC/-xuezhi-mytool.git

# 运行脚本
cd xuezhi-mytool
./git-manager.sh
```

> 将 `SKILL-GIT.md` 和 `git-manager.sh` 复制到你的项目根目录即可使用。
